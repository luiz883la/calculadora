Este é um projeto antigo que desenvolvi. Trata-se de uma calculadora simples, criada quando eu estava finalizando meu curso de JavaScript.

Sempre achei programação muito interessante e decidi publicar este projeto no GitHub para que outras pessoas possam usá-lo como inspiração ou até mesmo como base para desenvolver seus próprios projetos.

A calculadora foi construída de uma forma um pouco mais trabalhosa e, na minha opinião, ainda está um pouco "bagunçada". Acredito que o código ainda não está totalmente otimizado, principalmente na parte das equações e das funções da calculadora.

Pretendo continuar atualizando este projeto à medida que eu for aprendendo mais. Se você analisar o código, ficarei muito grato por sugestões, dicas e ideias de melhorias, especialmente para implementar funcionalidades que ainda estão faltando.
